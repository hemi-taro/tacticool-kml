# Changelog

## [1.2.4] - 2026-06-19

### Changed

- Set Japan GSI 2020.0 as the default magnetic variation mode
- Added clearer magnetic variation details in Object List
- Added Axis endpoint variation and inbound heading display
- Improved GSI range error messaging
- Improved Object List editing layout

## [1.2.0] - 2026-06-18

### Added

- Added local magnetic variation handling for direct Box and Arc geometry
- Added True heading mode
- Added clearer default names for Axis, SAM, Box, and Arc objects

### Improved

- Improved Object List coordinate and variation details
- Improved Custom Point / Line / Area generation
- Improved color editing controls
- Fixed bugs

## [1.1.4] - 2026-06-16

### Fixed

- Improved Object List and pending point reordering
- Fixed an issue where dragging stopped after moving one position

## [1.1.2] - 2026-06-16

### Added

- GeoJSON import and export
- Custom Point, Arc, and Box generation
- Object center display for SAM Rings and Boxes

### Improved

- Improved Object List editing and reordering
- Improved coordinate input and display
- Improved UI labels and usability
- Fixed bugs

## [1.0.0] - 2026-06-14

- First stable release
